# E-Commerce Website 🛒

A modern E-Commerce web application built with Angular.

## Features

- User Authentication (Login & Register)
- Home Page
- Product Listing
- Product Details
- Shopping Cart
- Wishlist
- Category Filtering
- Search Functionality
- Responsive Design
- Multi-language Support (English & Arabic)
- Loading Spinner
- Toast Notifications

## Technologies Used

- Angular
- TypeScript
- HTML5
- CSS3
- Bootstrap
- PrimeNG
- ngx-translate
- ngx-spinner
- SweetAlert2
- Swiper
